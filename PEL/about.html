About the project

This Project is a dynamic web database designed to catalog references of the PEL laboratory. The exact functionality required of the application as specified by my supervisor is summarised below:

The goal of the project is to create a database structure and web-user interface to allow a user to input paper references, query the database and modify existing paper references.

Input

Users must be able to enter search queries and submit them to the server for processing. They should be prompted to input this information.

Edit

Authorised users with extra privileges should be allowed to add, edit or delete references in the database.

Access:

Users should be able to query the database and generate lists of papers AND/OR links to these papers according to the following parameters:

  • Date range- return any paper in specified date range (month and year)
  • Author
  • First Author
  • Classification e.g. Journal/ International Conference/ Unrefereed/ Patent Conference/Journal in which Published
  • AND & OR combinations of the above should be allowed

A user should be prompted to input these parameters.

If possible the formatted data should be exported (e.g. XML, HTML, bibtex or csv)

The input and edit functionalities should be password protected.
