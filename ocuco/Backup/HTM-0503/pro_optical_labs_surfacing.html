This module is incorporated into the Order Entry program. When the operator has finished
inputting the prescription details, the Surfacing Instructions Module produces the surfacing
instructions.

Labman is designed to operate in a tele – sales environment, therefore the calculation
of the SI had to be quick -- less than two seconds. A summary of the results (base,
curves, substance & edge) is shown in the order entry screen. The full details are included
in the works docket. There is a facility to see how the SI is calculated on the screen.

Labman calculates the minimum size diameter required. If the lens falls into the
stock range and there is a lens in that diameter in stock then a stock lens is chosen. If a
stock lens is not chosen then the computer calculates the ideal base from which to
manufacture the lens; there is automatic slipping to match the other eye if the
prescriptions for both eyes are close.

Labman then uses this information as the basis to select the blank.If
there are blanks from different suppliers in stock then the system will choose according to
rules specified by you, e.g. choosing the most expensive in order to reduce the stock
valuation. There are options available to try other bases & diameters. The system will
automatically try to use a pair from the same supplier.
Surfacing instructions are available for the following lens forms:
  • Single Vision
  • Curve Top Bifocal
  • Flat Top Bifocal
  • Varifocal Lenses
  • Plus and Minus Lenticular
  • Round Segments
  • Executive Bifocals
  • Solid Bifocals
  • Trifocals
The above forms can also be calculated in an Aspheric format and in specified materials and
refractive indices.
There are three methods to obtain the frame shape :
  • Keyboard input (eye size, depth & longest length in the four quadrants),
  • Linking Labman to the drs Computing digitizer
  • Linking Labman to frame tracers.
Labman also has the facility to include different progressive lens designs for use in its calculations.
