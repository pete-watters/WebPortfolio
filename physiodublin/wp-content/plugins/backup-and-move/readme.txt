== Description ==

Backup And Move plugin allow blog administrators to create a complete backup of their blog and easy option for restore it on a different server, domain, location, etc .This plugin is very usefull for non technical users who wants to shift their wordpress blog.
This   plugin can make all the transitions of moving a wordpress blog ,   creating a complete backup and restoring a previous backup very simple   and smooth.
Save restore.php by <a href="http://plugins.svn.wordpress.org/backup-and-move/tags/restore.php" > clicking here</a>.
<ul>
  <li>A separate backup can be created and send to users's email id as attachment.</li>
  <li>User can administrate backups i.e delete , download and overwrite.</li>
  <li>To   restore a backup simply place .zip backup file in a directory along   with restore.php and visit restore.php from browser. Provide restore.php   with all required credentials and wait for plugin to restore .</li>
  <li>That's it start bloging.</li>
</ul>
<br />
P.S everything will be restored like blogs, posts, links , etc.

== Installation ==

1. Upload `backup_and_move` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it try creating a new backup.
